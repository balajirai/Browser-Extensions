# ChatGPT Molly Guard

A lightweight browser extension that prevents typing restricted keywords into the ChatGPT prompt. It helps keep prompt input clean by blocking forbidden terms before they are entered.

## Features

- Stops restricted keywords from being entered in the ChatGPT prompt
- Simple, focused protection for safer prompt creation
- Works directly in the typing flow

## Usage

1. Install the extension in your browser.
2. Open ChatGPT and start typing.
3. Restricted keywords are blocked as you type, preventing submission of disallowed content.

## Notes

This extension is intended to help avoid prompt injection and restricted wording by intercepting and disabling forbidden keyword input.
