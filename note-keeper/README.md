# Note Keeper — Quick in-browser notes

Note Keeper is a lightweight browser extension for jotting short, momentary notes without leaving your browser or opening another app.

Key features
- Quick notes: Add brief notes instantly while browsing.
- Always-on: Open as a popup from the toolbar—no separate app or notepad needed.
- Lightweight: Minimal UI focused on speed and convenience.

How to use
1. Install the extension in your browser.
2. Click the Note Keeper icon in the toolbar.
3. Type your note and save. Notes are available while browsing.

Privacy
- Notes are stored locally in the browser. No external servers are used by default.
